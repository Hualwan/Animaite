# Animate
为你的元素添加CSS3动画吧，让它们更加的生动丰富！<br/>

##介绍

你可以通过添加类名来实现动画效果，类名如下：

  * `bounce（弹跳）`
  * `flash（闪烁）`
  * `pulse（脉搏）`
  * `rubberBand（橡皮筋）`
  * `shake（抖动）`
  * `swing（摆动）`
  * `bounceIn（弹进）`
  * `bounceInDown（自上划入）`
  * `bounceInLeft（自左划入）`
  * `bounceInRight（自右划入）`
  * `bounceInUp（自下滑入）`
  * `bounceOut（弹出）`
  * `bounceOutDown（向下划出）`
  * `bounceOutLeft（向左滑出）`
  * `bounceOutRight（向右滑出）`
  * `bounceOutUp（向上划出）`
  * `flip（翻转）`
  * `flipInX（X轴翻转）`
  * `flipInY（Y轴翻转）`
  * `rollIn（滚进）`
  * `rollOut（滚出）`
  
例子:
```html
<h1 class="animated bounce">例子</h1>
```
